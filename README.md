# Notion Avatars

Create, Use and Download Notion Style Avatars ✨

I wanted to use existing Notion Avatar but it's broken ):

So I decided to make own version of it using Nuxt.js just for fun 😇

## Styles

- 😉 16 face shapes
- 👃🏼 14 noses
- 👄 20 mouths
- 👀 14 eyes
- 👁️ 16 eyebrows
- 🕶️ 14 glasses
- 💇‍♀️ 58 hairstyles
- 🎅🏼 16 beards
- 💋 13 facial details
- 💍 14 accessories
